Description:
Advanced Web Statistics (AWStats) is a free powerful and featureful web server logfile analyzer (Perl script) that shows you all your Web statistics including visits, unique visitors, pages, hits, rush hours, os, browser's versions, search engines, keywords, robots visits, broken links and more...Works with all major web servers (IIS 5.0+, Apache, ...) as a CGI and/or from command line.Distributed under GNU General Public License.
